# Other Services


## Workspaces

- Managed desktop service for Windows or Linux desktops

## App Stream

- Desktop application streaming service

## IOT Core

- Supports connecting IoT devices to AWS
- Serverless

## AppSync

- Store and sync data across web and mobile application
- Leverages GraphQL
- Offline data sync
- Integrates with Lambda and DynamoDB

## Amplify

- Tools and services to deploy full-stack web and mobile applications

## AWS Infrastructure Composer

- Visually design and build serverless applications

## Device Farm

- Managed service that allows you to test your apps against different browsers, mobile devices and tablets
- Runs test concurrently across multiple devices
- Ability to configure device settings

## AWS Backup

- Managed service to manage backups across AWS services
- On-demand and scheduled backups
- Supports point in time recovery

## Elastic Disaster Recovery

- Recover physical. virtual and cloud based servers
- Continous block replication for servers

## AWS DataSync

- Move large amounts of data from on-prem to the cloud
- Can syncronize to S3, EFS and FSx Windows
- Replication can be scheduled, daily, hourly, etc
- Replication tasks are incremental after first sync

## Cloud Migration Strategies

- 7 R's of cloud migration
  - Rehost (“Lift and Shift”)
  - Relocate (“Hypervisor-Level Lift and Shift”)
  - Replatform (“Lift and Reshape”)
  - Refactor (“Re-architect”)
  - Repurchase (“Drop and Shop”)
  - Retire
  - Retain (“Revisit”)

## Application Discovery Service

- Plan migrations by getting info about on-prem data centers
- Agentless discovery
- Agent based discovery

## AWS Migration Evaluator

- Helps build a business case for migration to AWS

## AWS Migration Hub

- Centralized location to track AWS migrations

## AWS Fault Injection Simulator

- Managed service for running fault injection experiments in AWS workloads
- Based on chaos engineering
- You can use pre-built templates for desired disruptions

## Step Functions

- Visual workflow to orchestrate lambda functions

## Ground Station

- Fully managed service that lets you control satellite communications, process data and scale operations
- Download satellite data via AWS VPC

## AWS Pinpoint

- Inbound/Outbound Marketing communication service
- Includes
  - Push notifications
  - Voice
  - In app messaging
  - SMS
