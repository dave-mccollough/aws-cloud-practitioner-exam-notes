# Advanced Identity

## AWS Secure Token Service (STS)

- Allows you to create temporary limited privilige credentials to AWS resources
- Short term credentials - you configure expiration

## AWS Cognito

- Identity service for web and mobile applications

## AWS Directory Services

- AWS Managed Microsoft Active Directory 
  - Create your own AD in AWS
- AD Connector
  - Integrates with on-prem AD
- Simple AD
  - AD compatible directory service

## AWS IAM Identity Center

- Single Sign on for all AWS accounts in AWS org
- SSO for business cloud apps
- EC2 Windows instances
- Built in identity store (IAM)
  - Integrates with 3rd party identity services
